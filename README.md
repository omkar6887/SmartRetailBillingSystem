# Smart Retail Billing System 

## Tech Stack
- ASP.NET MVC (.NET Framework)
- Entity Framework 6
- SQL Server
- Bootstrap

## Features
- Login & Role Based Access
- Barcode Billing System (POS)
- Cart Management
- Invoice Generation
- Stock Management
- Inventory Logs
- Dashboard Analytics

## Modules
- Authentication
- Products & Inventory
- Billing (POS)
- Reports & Invoice

## Database Setup
Run the SQL script inside the Database folder to create tables.
